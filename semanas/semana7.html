ESTILOS CSS CON REACT + VITE

En esta semana 7 hablaremos acerca de los estilos que se le da dentro de REACT

DEFINICIONES


ESTILOS CSS: Para añadir estilos a mi pagina tenemos dos opciones:

  • Crear un documento con .css
  • Crear un documento con nombre.module.css

En ambos casos se exporta la direccion a la pagina que queremos alterar con los estilos
COOMONS: Refiere a archivos comunes
Dentro de esta carpeta pueden estar los documentos con su css y crear alertas. Ejemplo:
  • NavBarH.jsx
  • NavBarH.module.css

Dentro de la carpeta COMMONS tambien puede estar las siguientes carpetas:
  • Footer
  • Header
  • Main

FEATURES

Dentro de esta carpeta encontraremos formularios con caracteristicas.

TAILWIND

Dentro de REACT + VITE tambien podemos añadir estilos que nos ofrece TAILWIND.

PROCEDIMIENTO

  • Colores en las fuentes
  • Posición en el menú
  • Alineación de los textos


RESULTADOS
