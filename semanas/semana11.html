NEXT JS

En esta semana 11 hablaremos acerca de Next JS

DEFINICIONES


NEXT JS: Es un framework de React utilizado principalmente para la creación de aplicaciones web modernas, con un enfoque en la facilidad de desarrollo, el rendimiento y la optimización para motores de búsqueda.

Algunos de sus servicio de renderizado son:
  • Server-Side Rendering (Pre renderizado)
  • Static Site Generation (Pre renderizado)
  • Client-Side Rendering.

Hablaremos un poco mas de Next Js
Renderización del lado del servidor y del lado del cliente:
Next.js soporta la renderización tanto del lado del servidor (SSR) como del lado del cliente (CSR). Esto significa que las páginas pueden ser renderizadas en el servidor y luego enviadas al cliente, lo que mejora la velocidad de carga inicial y optimiza la experiencia del usuario.

Enrutamiento basado en sistema de archivos:
Next.js utiliza un enrutamiento basado en el sistema de archivos. Esto significa que las rutas de las páginas se definen según la estructura de carpetas y archivos en el proyecto, lo cual facilita la organización y mantenimiento de las rutas.

API Routing:
Next.js incluye soporte para definir endpoints de API dentro del mismo proyecto, lo que facilita la creación de servicios backend personalizados sin necesidad de configuraciones adicionales.
